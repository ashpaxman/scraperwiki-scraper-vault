Some working scrapers that are worth looking at, among the many that I have created.

Outer space affairs

The United Nations Office of Outer Space Affairs runs a register of man-made objects.

outer_space_objects_scrapedownloader
Download all the index pages from the database
outer_space_objects_parsecollector
Load from the database above and parse into a clean database. Still needs to do the country iso identification
outer_space_objects_view
Curious circle visualization of number of satellite per country.

North sea oil wells

DECC have a database of all oil wells drilled since 1964 which is easy to scrape.

northsea_oilwell_scrapedownload
Does the download of each page into the database
northsea_oilwell_parse
Here we parse the downloaded data into clean data in a database
northsea_oilwell_server
Serves interrogations from the frontend of the database.
northsea_oilwell_view
Crude map view that slightly animates the drilling in the sea for each year

Sqlite technology

We can create a UI that allows for easy interrogation of any database kept by a scraper

general_sqlite_view
Front end HTML page that allows building up the SQL command from its components
general_sqlite_interrogator
This provides the backend that executes the SQL query against the database

Cave mapping

cave-svg-viewer
Preview the map using jquery.svg technology. Under development
cave-svg-server
Serve out the map for the front end to use. Needs to handle regions of interest
survex-cave-data-loader
Intended as a complete parsing for a svx file
