New Mexico State Audits

Opinion changes

The query is

SELECT j1.nodeId AS nodeId, (j2.time_scraped - j1.time_scraped) AS time_diff,
j1.Opinion as old, j2.Opinion as new FROM opinions j1 INNER JOIN opinions j2
ON (j1.nodeId = j2.nodeId) WHERE j1.Opinion != j2.Opinion
Preview this query.
